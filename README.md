# システム
ESP32からサーバーへUDP通信で取得データを送信する。
その取得データから押されたかどうかを判別する。
botに量が含まれた文字を送信すると、シャンプーの残りの残量の画像が送信される

# ソフトウェア
Python 3.7.3
Processing 3.5.3
Arduino IDE 1.8.9
